# Markdown 功能

Docusaurus 支持 **[Markdown](https://daringfireball.net/projects/markdown/syntax)** 以及一些**额外的功能**。

## Front Matter

Markdown 文档顶部有一个称为 [Front Matter](https://jekyllrb.com/docs/front-matter/) 的元数据部分：

```text title="my-doc.md"
// highlight-start
---
id: my-doc-id
title: 文章title
description: 文章描述
slug: /my-custom-url
---
// highlight-end

## Markdown heading

Markdown text with [links](./hello.md)
```

## 链接

支持常规的 Markdown 链接，可以使用 URL 路径或相对文件路径。

```md
Let's see how to [Create a page](/create-a-page).
```

```md
Let's see how to [Create a page](./create-a-page.md).
```

**结果:** 跳转到 [创建一个页面](./create-a-page.md).

## 图片

支持常规的 Markdown 图片。

可以使用绝对路径来引用静态目录中的图片 (`static/img/docusaurus.png`)：

```md
![Docusaurus logo](/img/docusaurus.png)
```

![Docusaurus logo](/img/docusaurus.png)

也可以用相对路径引用图片

```md
我们目前在docs/tutorial-basics 所以要往上层翻两次
![Docusaurus logo](../../static/img/docusaurus.png)
```

![Docusaurus logo](../../static/img/docusaurus.png)

## 代码块

支持带有语法高亮的 Markdown 代码块

    ```jsx title="src/components/HelloDocusaurus.js"
    function HelloDocusaurus() {
        return (
            <h1>Hello, Docusaurus!</h1>
        )
    }
    ```

```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>Hello, Docusaurus!</h1>;
}
```

## 提示

Docusaurus 有一种特殊语法来创建提示和警告：

    :::tip My tip
    
    good
    
    :::
    
    :::danger Take care
    
    This action is dangerous
    
    :::

:::tip 小tips
good 👍
:::

:::danger Take care
危险 ❌
:::

## MDX 和 React 组件

[MDX](https://mdxjs.com/) 可以使你的文档更加**互动**，并允许在 Markdown 中使用任何 **React 组件**：

```jsx
export const Highlight = ({children, color}) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: '20px',
      color: '#fff',
      padding: '10px',
      cursor: 'pointer',
    }}
    onClick={() => {
      alert(`You clicked the color ${color} with label ${children}`)
    }}>
    {children}
  </span>
);

This is <Highlight color="#25c2a0">Docusaurus green</Highlight> !

This is <Highlight color="#1877F2">Facebook blue</Highlight> !
```
